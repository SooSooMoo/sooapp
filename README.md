# ⚽ 観戦旅行プランAI

LangChain + GPT-4 を使った、スポーツ観戦旅行プラン自動生成アプリです。

## 💡 概要

ユーザーが指定したスポーツ、国、年、月に基づいて、DuckDuckGo検索を活用して試合日程を調査し、GPT-4が5日間の旅行プランを自動生成します。

## 🧪 使用技術

- Python
- Streamlit
- OpenAI GPT-4 (要APIキー)
- LangChain
- DuckDuckGo検索ツール
